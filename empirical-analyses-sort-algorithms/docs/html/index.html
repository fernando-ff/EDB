Empirical analyses

A simple implementation of the following algorithms for array of integers:

  • insertion sort
  • selection sort
  • bubble sort
  • shell sort
  • quick sort
  • merge sort
  • radix sort
Authors
Vitor Hugo e Fernando Ferreira
Date
Set, 2020.

Starting

clone this project

$ git clone https://projetos.imd.ufrn.br/fernandoff/edb1-empirical-analysis.git

Access

$ cd edb1-empirical-analysis

to compile

$ mkdir build && cd build $ cmake ../ $ make

run where the data collection occurs

$ ./timing_template

run the tests

$ ./run_tests
